# OCB12-PLD Firmware

_In-development unofficial MSX2+ firmware for OneChipBook-12 systems._
## Supported Devices

**1st Gen**
- OneChipBook-12 (a.k.a MSXBOOK)
- 1chipMSX
- Zemmix Neo (KR)
- Zemmix Neo BR
- SX-1 (regular)
- SX-1 Mini/Mini+

> **Note:** Not tested on anything other than OneChipBook-12.
